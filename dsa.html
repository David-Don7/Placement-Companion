Two Sum

Easy Arrays

Problem Statement

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target. You may assume that each input would have exactly one solution, and you may not use the same element twice.

💡 Solution Approach

Brute Force: Use two nested loops to check every pair — O(n²).

Optimal (Hash Map): Iterate through the array, for each element check if (target − element) exists in a hash map. If yes, return the indices. Otherwise, store the element. Time: O(n), Space: O(n).
